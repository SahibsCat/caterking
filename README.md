# Sahibs Catering - Setup Guide

This project is a premium catering booking platform built with a cross-platform stack (Node.js, React, MongoDB).

## 🚀 Quick Start (Windows/Linux/macOS)

1. **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) (v18+) and [MongoDB](https://www.mongodb.com/) installed.
2. **Clone/Copy Project**: Copy this entire folder to your local machine.
3. **Backend Setup**:
   ```bash
   cd backend
   npm install
   npm run dev
   ```
4. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
